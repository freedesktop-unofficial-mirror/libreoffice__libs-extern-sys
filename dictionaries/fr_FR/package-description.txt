Quatre dictionnaires français existent:  
— «Moderne» : le français avec les mots réformés passés dans l’usage. [recommandé]  
— «Classique» : le français d’avant la réforme de l’orthographe.  
— «Réforme 1990» : le français selon la réforme de l’orthographe.  
— «Classique & Réforme 1990» : toutes les graphies sont reconnues.
